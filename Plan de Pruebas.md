Se realizó un plan de pruebas para verificar el correcto funcionamiento del sitio web mediante pruebas manuales que simulan el uso real. Se evaluaron la carga inicial, la navegación entre secciones, el funcionamiento de botones y la validación de formularios. Además, se comprobó la adaptabilidad en distintos dispositivos y la correcta organización visual para garantizar una buena experiencia de usuario.
